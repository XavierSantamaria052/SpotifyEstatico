# Arquitectura del proyecto

La aplicación sigue una arquitectura frontend estática basada en componentes visuales.

```text
┌──────────────────────────────┐
│          HTML UI             │ ← Estructura principal
├──────────────────────────────┤
│            CSS               │ ← Estilos y layout
├──────────────────────────────┤
│        Responsive UI         │ ← Adaptación móvil
├──────────────────────────────┤
│      Componentes Visuales    │ ← Sidebar, Player, Cards
├──────────────────────────────┤
│         Assets Media         │ ← Imágenes e íconos
└──────────────────────────────┘
```
